## Readme
### 初始化前端工程 - react + typescript
```
npm create vite@latest destination --template react-ts
```

### 安装tdesign-react
```shell
npm install tdesign-react
```

### 安装前端与后端配合组件
```shell
npm install axios
```

## 开发倒计时vercel使用
**环境变量**
```
VITE_COUNTDOWN_NAME=2026年12月18日    # 倒计时目标名称
VITE_COUNTDOWN_DATE=2026-12-18T00:00:00+08:00  # ISO 8601 目标时间
VITE_COUNTDOWN_TITLE=倒计时              # 浏览器标签页标题
```
